🎮 Clout Clash – Higher Lower Game

Clout Clash is a command-line Python game inspired by the classic Higher–Lower concept. Players are presented with two public figures and must guess who has more social media followers. Each correct answer increases the score — one wrong guess ends the game.

🚀 Features

Randomized account comparisons

Score tracking system

Continuous gameplay until incorrect guess

Clean terminal interface with ASCII art

Beginner-friendly Python structure

🧠 Concepts Practiced

This project reinforces core Python fundamentals:

Functions

Loops (while)

Conditional statements

Dictionaries

Random module usage

User input handling

Basic game logic

📂 Project Structure
├── main.py       # Game logic
├── art.py        # ASCII art (logo & vs)
├── data.py       # Follower dataset
└── README.md


🎯 How to Play

You will see two accounts labeled A and B.

Type A or B to guess who has more followers.

Each correct guess increases your score.

One wrong guess ends the game.
